1. 프로젝트의 목적
2. 프로그램의 구조

4. 프로그램의 사용 방법

1. 프로젝트의 목적
이 프로젝트는 C언어로 주문관리 시스템을 개발해서 상품, 고객, 주문을 관리하는 것을 간단하게 해낼 수 있는 것을 목적으로 둡니다.

2. 프로젝트의 구조
이 프로젝트로 만들어진 프로그램은 mainmenu.exe, order.h, product.h, customer.h로 이루어져 있습니다.
2-1. mainmenu.exe
mainmenu.exe는 메인 함수에서 프로그램이 시작될 때, 주문 관리, 상품 관리, 고객 관리 시스템을 선택할 수 있는 주문 관리 시스템을 소개합니다. 이때, order.h, product.h, customer.h 세 개의 헤더 파일로 분리된 각 시스템의 기능만을 선택하여 실행합니다. 사용자가 종료를 선택할 때까지 이러한 과정을 반복합니다.
이를 통해 각 시스템의 기능이 모듈화되어 있으며, 메인 함수를 중심으로 프로그램의 흐름을 제어하도록 설계되어 코드의 유지보수와 확장이 용이하며 코드의 가독성도 향상되었습니다.
2-2 product.h
product.h는 상품 관리를 위한 헤더 파일입니다. 따라서  상품 정보는 상품 번호, 상품명, 가격으로 구성된 구조체 멤버 변수로 저장됩니다. 또한, 상품 관리 항목에 진입하면 상품 관리 메뉴가 표시되고 사용자로부터 원하는 항목을 선택받아 해당 기능을 수행하는 함수를 호출합니다. 상품 추가, 수정, 삭제, 목록 출력을 위한 함수들은 각각의 기능을 수행하고 입력된 정보를 txt 파일로 저장합니다. 
이를 통해 코드의 가독성이 향상되고 유지 보수와 확장이 용이하며, 파일을 활용하여 상품 정보를 영구적으로 저장하고 관리할 수 있습니다.

2-3 order.h
order.h는 주문 관리를 위한 헤더 파일입니다. 주문 정보는 주문 번호, 상품명, 가격, 고객명, 전화번호로 구성된 구조체 멤버 변수로 저장됩니다. 주문 관리 항목에 진입하면 주문 관리 메뉴가 표시되고 사용자로부터 원하는 항목을 선택받아 해당 기능을 수행하는 함수를 호출합니다. 주문 추가, 수정, 제거, 목록 출력, 상세 내역 출력을 위한 함수들은 각각의 기능을 수행하고 입력된 정보를 txt 파일로 저장합니다. 
이를 통해 코드의 가독성이 향상되고 유지보수와 확장이 용이하며, 파일을 활용하여 주문 정보를 영구적으로 저장하고 관리할 수 있습니다.

2-4 customer.h
customer.h는 고객 관리를 위한 헤더파일입니다. 고객 정보는 고객 번호, 고객명, 전화번호로 구성된 구조체 멤버 변수로 저장됩니다. 사용자가 고객 관리 항목에 접근하면 고객 관리 메뉴가 표시되고, 사용자는 원하는 항목을 선택하여 해당 기능을 수행하는 함수를 호출합니다. 추가, 수정, 제거, 목록 출력을 위한 함수들은 각각의 기능을 수행하며, 입력된 정보를 txt 파일로 저장합니다. 이를 통해 코드의 가독성이 향상되고 유지보수와 확장이 용이하며, 파일을 활용하여 고객 정보를 영구적으로 저장하고 관리할 수 있습니다.

3. 프로그램의 주요 기능
프로그램의 주요 기능은 목적을 달성하기 위해 항목이동, 상품관리, 고객관리, 주문관리와 같이 구성되어 있습니다.

3-1. 항목 이동 기능
원하는 항목으로 이동하는 기능: 숫자를 입력하여 원하는 항목으로 이동할 수 있습니다.

3-2. 상품 관리 기능
상품 추가: 새로운 상품을 등록합니다.
상품 수정: 등록된 상품의 정보를 수정합니다.
상품 삭제: 등록된 상품을 삭제합니다.
상품 목록 확인: 등록된 상품들의 목록을 확인합니다.

3-3. 고객 관리 기능
고객 추가: 새로운 고객을 등록합니다.
고객 수정: 등록된 고객의 정보를 수정합니다.
고객 삭제: 등록된 고객을 삭제합니다.
고객 목록 확인: 등록된 고객들의 목록을 출력합니다.

3-4. 주문 관리 기능
주문 추가: 새로운 주문을 등록합니다.
주문 수정: 등록된 주문의 정보를 수정합니다.
주문 삭제: 등록된 주문을 삭제합니다.
주문 목록 출력: 등록된 주문의 목록을 출력합니다.
주문 상세 출력: 등록된 주문의 상세한 내역을 출력합니다.

4. 프로그램의 사용 방법
프로그램의 사용 방법은 아래와 같습니다.

4-1. 프로그램 실행 후 원하는 메뉴로 이동하기
먼저, 프로그램을 실행하게 되면 원하는 관리 작업을 수행할 수 있는 메뉴를 안내하는 글이 표시됩니다. 예:"상품 관리를 원하시면 1, 고객 관리를 원하시면 2 ..."
이때, 원하는 항목의 번호를 입력하고 엔터키를 누르면 해당 관리 항목으로 이동하게 됩니다.

4-2. 상품 관리 항목
상품 관리 항목은 상품 추가, 상품 수정, 상품 삭제, 상품 목록 출력을 할 수 있습니다.
상품관리 항목으로 이동하는 방법은 맨 처음 실행 화면에서 1을 입력하고 이동하실 수 있습니다.
4-2.1 상품 추가
상품 관리 항목에서 상품을 추가하려고 할 때는 다음과 같이 하면 됩니다.
1 > 상품번호 입력 > 상품명 입력 > 상품 가격 입력
이후 추가를 완료하게 되면 "상품이 추가되었습니다."라고 메시지가 출력됩니다.
4-2.2 상품 수정
상품 관리 항목에서 상품명과 가격을 수정하려고 할 때는 다음과 같이 하면 됩니다.
2 > 수정할 상품번호 입력 > 수정할 상품명 입력 > 수정할 가격 입력
이때, 상품번호를 통해 기존 상품의 이름과 가격을 수정하므로, 기존의 상품번호를 입력하고 새로 수정할 이름과 가격을 입력해야 합니다.
이후 수정을 완료하면 "상품이 수정되었습니다."라고 메시지가 출력됩니다.
4-2.3 상품 삭제
상품 관리 항목에서 등록된 상품을 삭제하고 싶다면 다음과 같이 하면 됩니다.
3 > 삭제할 상품의 상품번호 입력
이후 삭제를 완료하면 "상품이 삭제되었습니다."라고 메시지가 출력됩니다.
4-2.4 상품 목록 출력
상품 관리 항목에서 등록된 상품 목록을 확인하려면 4번을 입력하면 됩니다.
이후 등록된 상품 목록이 출력됩니다.
4-2.5 메인 메뉴로 나가기
상품 관리 항목에서 볼 일이 없거나, 다 끝냈다면 5번을 입력하면 됩니다.
이후 메인 메뉴로 이동됩니다.

4-3. 고객 관리 항목
고객 관리 항목은 고객 추가, 고객 수정, 고객 삭제, 고객 목록 출력을 할 수 있습니다.
고객 관리 항목으로 이동하는 방법은 맨 처음 실행 화면에서 2를 입력하고 이동하실 수 있습니다.
4-3.1 고객 추가
고객 관리 항목에서 고객을 추가하려고 할 때는 다음과 같이 하면 됩니다.
1 > 고객 번호 입력 > 고객 이름 입력 > 고객의 전화번호 입력
이후 추가를 완료하게 되면 "고객이 추가되었습니다."라고 메시지가 출력됩니다.
4-3.2 고객 수정
고객 관리 항목에서 고객을 수장하려고 할 때는 다음과 같이 하면 됩니다.
2 > 수정할 고객 번호 입력 > 수정할 고객 이름 입력 > 수정할 고객의 전화번호 입력
이때, 고객 번호를 통해 기존 고객의 이름과 전화번호를 수정하므로, 기존의 고객 번호를 입력하고 새로 수정할 이름과 전화번호를 입력해야 합니다.
이후 수정을 완료하게 되면 "고객 정보가 수정되었습니다."라고 메시지가 출력됩니다.
4-3.3 고객 삭제
고객 관리 항목에서 고객을 삭제하려고 할 때는 다음과 같이 하면 됩니다.
3 > 삭제할 고객 번호 입력
이후 삭제를 완료하게 되면 "고객이 삭제되었습니다."라고 메시지가 출력됩니다.
4-3.4 고객 목록 출력
고객 관리 항목에서 등록된 고객 목록을 확인하고 싶다면 4번을 입력하면 됩니다.
이후 등록된 고객 목록이 출력됩니다.

4-4. 주문 관리 항목
주문 관리 항목은 주문 추가, 주문 수정, 주문 삭제, 주문 목록 출력, 주문 상세 출력을 할 수 있습니다.
주문 관리 항목으로 이동하는 방법은 맨 처음 실행 화면에서 3을 입력하고 이동하실 수 있습니다.
4-4.1 주문 추가
주문 관리 항목에서 주문을 추가하려고 할 때는 다음과 같이 하면 됩니다.
1 > 주문 번호 입력 > 상품명 입력 > 가격 입력 > 주문자 입력 > 주문자의 전화번호 입력
이후 추가를 완료하게 되면 "주문이 추가되었습니다."라고 메시지가 출력됩니다.
4-4.2 주문 수정
주문 관리 항목에서 주문을 수정하려고 할 때는 다음과 같이 하면 됩니다.
2 > 수정할 주문 번호 입력 > 수정할 상품명 입력 > 수정할 가격 입력 > 수정할 주문자 입력 > 수정할 주문자의 전화번호 입력
이때, 주문번호를 통해 기존 상품명, 가격, 주문자, 전화번호를 수정하므로, 기존의 주문번호를 입력하고 새로 수정할 상품명, 가격, 주문자, 전화번호를 입력해야 합니다.
이후 수정을 완료하게 되면 "주문 정보가 수정되었습니다."라고 메시지가 출력됩니다.
4-4.3 주문 삭제
주문 관리 항목에서 주문을 삭제하려고 할 때는 다음과 같이 하면 됩니다.
3 > 삭제할 주문 번호 입력
이후 삭제를 완료하게 되면 "주문이 삭제되었습니다."라고 메시지가 출력됩니다.
4-4.4 주문 목록 출력
주문 관리 항목에서 등록된 주문 목록을 확인하고 싶다면 4번을 입력하면 됩니다.
이후 등록된 주문 목록이 출력됩니다.
4-4.5 주문 상세 출력
주문 관리 항목에서 등록된 주문의 상세 정보를 확인하고 싶다면 5번을 입력하면 됩니다.
이후 등록된 주문 목록의 상세 정보가 출력됩니다.
4-4.6 메인메뉴로 나가기
주문 관리 항목에서 볼 일이 없거나, 다 끝냈다면 6번을 입력하면 됩니다.

4-5. 프로그램 종료
프로그램에서 볼 일이 없다면 4를 입력하고 엔터키를 눌러서 끌 수 있게 해뒀으며,
여느 프로그램들과 같이 창의 오른쪽 위에 위치한 "X" 표시를 클릭해서 끌 수도 있습니다.
